从被期待的人，到自我觉醒的存在

A journey from expectations to self-awakening.
中文

本项目是一场从“被期待的人”到“自我觉醒的存在”的旅程。 以心理学为方法，以哲学为骨架，帮助人从 经验 → 意识 → 行动 → 自我创造 的路径中， 重建与自我、他人、时间的关系。

English

This project is a journey from being “the expected one” to becoming a self-awakened presence. Using psychology as method and philosophy as backbone, it supports a path from experience → awareness → action → self-creation, rebuilding one’s relationship with self, others, and time.
